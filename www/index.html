    Introduction

    Welcome to the Marcus Aurelius Quotes for Mobius API! You can use our API to receive a random Marcus Aurelius quote.

    This is an HTTP only API, so you can use either curl or a browser to access it.

    If you would like to create something like this, there is documentation here: https://medium.com/@billzajac/mobius-with-python-and-google-appenine-68e0cf7e4060

    And example code here: https://github.com/billzajac/mobius-marcus-aurelius

    This API documentation page was created with DocuAPI, a multilingual documentation theme for the static site generator Hugo.

    Authentication

    To authorize, simply try the quote URL below in a browser and the Mobius Network will authenticate you and request authorizeation.

    You can register your email at Mobius Network.

    Quotes

    Get a quote

    curl "https://mobius-marcus-aurelius.appspot.com/quote?email=YOUR_EMAIL"

    Make sure to replace YOUR_EMAIL with your (Mobius registered) email. The Mobius Network will request authorization to charge your account.

    The above command returns JSON structured like this:

    [
  {
    "quote": "When you arise in the morning, think of what a precious privilege it is to be alive - to breathe, to think, to enjoy, to love.", 
    "num_credits": "4.0",  
    "charged_credits": "0.0"
  }
]
    You must replace YOUR_EMAIL with your (Mobius registered) email.

    This endpoint retrieves a quote.

    HTTP Request

    GET /quote

    Query Parameters

    Parameter Default Description
    email N/A Email address to bill
